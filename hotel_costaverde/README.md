# Hotel Costa Verde

Sistema web inicial en PHP 8 con Oracle OCI8, Bootstrap 5 y arquitectura simple para sitio público y panel administrativo.

## Accesos
- Sitio público: /public/index.php
- Panel administrativo: /admin/login.php
- Credenciales de ejemplo: usar el correo registrado en la tabla USUARIOS y su contraseña.

## Verificación realizada
- Conexión OCI8 comprobada con éxito.
- Consultas parametrizadas verificadas contra Oracle.
- Sintaxis PHP validada sin errores.
